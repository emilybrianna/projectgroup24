# SMARTFASH – AI-Powered Outfit Discovery Platform

SMARTFASH is an AI-powered mobile application developed as a Diploma in Computer Science Final Year Project. The platform helps users discover suitable outfits through personalized AI recommendations while providing a complete online fashion shopping experience.

The application combines artificial intelligence, personalized styling, virtual wardrobe management, and e-commerce features into a single mobile platform.

---

# Overview

SMARTFASH is designed for three types of users:

* Customers – Discover outfits, shop fashion products, and receive AI-powered recommendations.
* Staff – Manage products, monitor stock, and generate reports.
* Administrators – Manage users, products, orders, analytics, and overall system operations.

The project was developed using the Scrum methodology, with features implemented across multiple development sprints.

# AI Features

## AI Outfit Recommendation

Provides personalized outfit suggestions based on:

* User preferences
* Fashion style
* Weather conditions

## AI Fashion Stylist

Interactive chatbot that provides:

* Fashion advice
* Outfit suggestions
* Product recommendations

## Body Shape Analyzer

Analyzes full-body images and recommends clothing that complement the user's body shape.

## Skin Tone Analyzer

Analyzes skin tone and suggests suitable clothing colors.

## Virtual Closet

Users can upload their own clothing items and receive AI-generated outfit combinations.

## Smart Packing Assistant

Generates personalized packing checklists and outfit suggestions based on:

* Travel destination
* Trip duration
